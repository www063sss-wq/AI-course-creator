# 课前准备清单

## 课程信息
- **课程名称**：企业 AI 落地第一课：从 0 到 1 搭建智能客服
- **课程时长**：3 小时
- **目标学员**：企业技术负责人、产品经理、客服主管、企业决策者
- **预计人数**：20-30 人

---

## 一、课前 2 周准备

### 1.1 场地确认
- [ ] 确认培训场地（会议室/培训室）
- [ ] 场地容量确认（容纳 20-30 人）
- [ ] 场地布局（小组式/课桌式）
- [ ] 空调、通风、照明检查

### 1.2 设备确认
- [ ] 投影仪和屏幕（亮度、分辨率）
- [ ] 音响系统（麦克风、音箱）
- [ ] 讲师电脑（备用电脑）
- [ ] 网络环境（WiFi 密码、带宽）
- [ ] 电源插座（每组至少 2 个）

### 1.3 学员通知
- [ ] 发送课程通知邮件
- [ ] 发送预习材料
- [ ] 发送环境安装指南
- [ ] 收集学员背景信息
- [ ] 收集学员期望和问题

---

## 二、课前 1 周准备

### 2.1 教学材料准备
- [ ] 打印课程讲义（人手一份）
- [ ] 打印实践操作指南（人手一份）
- [ ] 打印工具模板（人手一份）
- [ ] 准备 PPT（最终版）
- [ ] 准备案例资料

### 2.2 环境准备
- [ ] 搭建演示环境
- [ ] 准备代码示例
- [ ] 测试所有工具和环境
- [ ] 准备备用方案（网络故障等）

### 2.3 物料准备
- [ ] 姓名牌/桌签
- [ ] 笔和笔记本
- [ ] 评估表格
- [ ] 课程反馈问卷
- [ ] 茶歇安排（如有）

---

## 三、课前 1 天准备

### 3.1 场地布置
- [ ] 桌椅摆放（小组式，每组 4-5 人）
- [ ] 投影和音响测试
- [ ] 网络测试
- [ ] 温度调节
- [ ] 指示牌放置

### 3.2 材料分发
- [ ] 每个座位放置讲义包
- [ ] 放置姓名牌
- [ ] 放置笔和笔记本
- [ ] 放置评估表格

### 3.3 最终检查
- [ ] 讲师电脑充电
- [ ] PPT 播放测试
- [ ] 演示环境最终测试
- [ ] 备用方案确认
- [ ] 助教分工确认

---

## 四、课前 1 小时准备

### 4.1 设备开启
- [ ] 开启投影和音响
- [ ] 开启讲师电脑
- [ ] 连接网络
- [ ] 播放欢迎页面

### 4.2 环境确认
- [ ] 温度适宜
- [ ] 光线适宜
- [ ] 通风良好
- [ ] 整洁有序

### 4.3 讲师准备
- [ ] 熟悉场地
- [ ] 测试麦克风
- [ ] 准备开场
- [ ] 调整状态

---

## 五、学员课前准备（邮件发送）

### 5.1 预习材料
- 课程大纲
- 智能客服基础概念（可选）
- 行业案例阅读

### 5.2 环境安装
- Python 3.8+ 安装
- VS Code 或类似编辑器
- Git 安装（可选）
- 网络环境确认

### 5.3 心理准备
- 带上企业实际问题
- 准备分享经验
- 保持开放心态

---

## 六、紧急情况预案

### 6.1 网络故障
- 准备离线版本材料
- 准备 4G/5G 热点
- 调整课程顺序（先理论后实践）

### 6.2 设备故障
- 备用电脑准备
- 备用投影仪确认
- 打印版 PPT 准备

### 6.3 时间调整
- 弹性时间模块
- 可压缩内容标识
- 休息时间安排
