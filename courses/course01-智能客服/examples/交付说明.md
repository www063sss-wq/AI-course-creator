## 5️⃣ API 集成示例代码 🔌

### 订单查询 API 集成

```python
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/api/order/<order_id>')
def query_order(order_id):
    """查询订单状态"""
    order_data = {
        'order_id': order_id,
        'status': '已发货',
        'product': 'iPhone 15 黑色 128G',
        'logistics': {
            'company': '顺丰快递',
            'tracking_no': 'SF123456',
            'status': '运输中',
            'estimated_delivery': '明天'
        }
    }
    return jsonify(order_data)

if __name__ == '__main__':
    app.run(debug=True)
```

### 运行演示

```bash
# 启动 API 服务
```
