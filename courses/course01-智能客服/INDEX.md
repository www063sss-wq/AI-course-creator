## 使用指南

### 课程负责人
1. 阅读 `core/` 了解课程全貌
2. 使用 `delivery/` 执行授课
3. 参考 `docs/reference/` 理解设计理念
4. 查看 `examples/` 了解实战示例

### 讲师
1. 精读 `core/01-课程大纲.md` 掌握课程结构
2. 熟悉 `core/02-教学方法.md` 掌握教学方法
3. 使用 `delivery/02-授课流程手册.md` 执行授课
4. 使用 `support/01-讲师讲稿.md` 进行授课
5. 演示 `examples/ecommerce/` 中的实战代码

### 助教
1. 阅读 `delivery/02-授课流程手册.md` 了解职责
2. 使用 `delivery/01-课前准备清单.md` 准备物料
3. 协助执行 `delivery/03-课后跟进手册.md`
4. 准备 `examples/` 中的代码演示环境

### 学员
1. 使用 `examples/` 中的实战代码示例
2. 参考 `support/` 中的资料
3. 遵循 `delivery/03-课后跟进手册.md` 的跟进计划
4. 运行 Web 聊天演示体验智能客服
