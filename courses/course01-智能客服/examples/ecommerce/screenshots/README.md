由于系统限制，无法自动生成截图。请手动访问以下地址查看实时演示：

**访问地址**: http://localhost:5001

## 🎨 界面特点

### 1. 聊天窗口设计
- **现代化 UI**: 渐变紫色背景，圆角卡片式设计
- **响应式布局**: 800x600 固定尺寸，适配各种屏幕
- **美观的对话气泡**: 
  - 用户消息：紫色渐变，右对齐
  - 机器人消息：白色背景，左对齐

### 2. 功能特性
- **快捷回复按钮**: 4 个预设问题，一键提问
  - 产品怎么用
  - 价格咨询
  - 查询订单
  - 退货流程
- **输入框**: 支持回车发送
- **发送按钮**: 点击发送，带动画效果
- **输入指示器**: 三个跳动的圆点，显示"正在输入..."

### 3. 交互体验
- **自动滚动**: 新消息自动滚动到底部
- **禁用状态**: 发送时按钮禁用，防止重复提交
- **加载状态**: 显示输入指示器，提升用户体验

## 📋 手动截图步骤

### 方法 1: 使用 macOS 截图
1. 打开浏览器访问 http://localhost:5001
2. 按 `Cmd + Shift + 4` 选择区域截图
3. 或按 `Cmd + Shift + 5` 打开截图工具

### 方法 2: 使用浏览器开发者工具
1. 打开 Chrome/Edge 浏览器
2. 按 `F12` 打开开发者工具
3. 按 `Cmd + Shift + P` 打开命令面板
4. 输入 "screenshot" 选择 "Capture full size screenshot"

### 方法 3: 使用在线截图工具
访问截图网站，输入 URL 进行截图
