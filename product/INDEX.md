## 使用指南

### 课程负责人
1. 阅读 `product/core/` 了解课程全貌
2. 使用 `product/delivery/` 执行授课
3. 参考 `docs/reference/` 理解设计理念

### 讲师
1. 精读 `01-课程大纲.md` 掌握课程结构
2. 熟悉 `02-教学方法.md` 掌握教学方法
3. 使用 `02-授课流程手册.md` 执行授课
4. 参考 `PPT 课件` 和 `案例库` 准备课程

### 助教
1. 阅读 `02-授课流程手册.md` 了解职责
2. 使用 `01-课前准备清单.md` 准备物料
3. 协助执行 `03-课后跟进手册.md`

### 学员
1. 使用 `product/tools/` 中的模板
2. 参考 `product/support/` 中的资料
3. 遵循 `03-课后跟进手册.md` 的跟进计划

## 版本管理

### 版本规则
- **主版本号**：重大重构（如 v1.0 → v2.0）
- **次版本号**：内容更新（如 v2.0 → v2.1）
- **修订号**：勘误修正（如 v2.1.0 → v2.1.1）

### 更新流程
1. 评估变更影响
2. 更新相关文档
3. 更新版本号
4. 更新依赖文档
5. Git 提交说明
